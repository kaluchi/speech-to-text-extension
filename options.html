Настройки голосового ввода

Вы можете получить API ключ, зарегистрировавшись на elevenlabs.io

Выберите язык распознавания для повышения точности.

При включении в распознанном тексте будут отмечены паузы, смех и другие звуковые события.

Выберите уровень детализации временных меток в транскрипции.

Включите для определения кто и когда говорит в записи.

Укажите максимальное количество говорящих (от 1 до 32). Оставьте пустым для автоопределения.

Добавьте ключевые слова и их веса (от -10 до 10) для улучшения распознавания.

Выберите конкретный микрофон для ускорения запуска записи. Список обновится при открытии настроек.

При включении будет воспроизводиться запись параллельно с отправкой на сервер.

При включении во время записи будет отображаться цветовая индикация (жёлтая при инициализации, зелёная при записи).
