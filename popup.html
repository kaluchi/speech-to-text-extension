Голосовой ввод с ElevenLabs

Расширение позволяет использовать распознавание речи через ElevenLabs API для голосового ввода текста в любое текстовое поле на веб-странице.

Инструкция по использованию:

  • Нажмите Ctrl дважды быстро и удерживайте
  • Говорите, пока удерживаете клавишу
  • Отпустите клавишу для завершения записи
  • Текст будет вставлен в активное поле ввода
Расширение активно
Разработано с использованием технологии ElevenLabs для распознавания речи.
