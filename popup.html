Voice Input with ElevenLabs

This extension allows you to use speech recognition through ElevenLabs API for voice input in any text field on a web page.

How to use:

  • Press Ctrl twice quickly and hold
  • Speak while holding the key
  • Release the key to finish recording
  • Text will be inserted into the active input field
Extension active
Developed using ElevenLabs speech recognition technology.
